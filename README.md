2018년 졸업작품으로 만들었던 아두이노 한자 게임입니다.
아두이노의 자이로/가속도 센서와 5버튼 모듈로 캐릭터를 조작하도록 만들었고 게임의 형태로 한자를 학습하도록 만들었습니다. 게임은 탐험/슈팅 장르입니다.
